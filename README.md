# Delete-temporary-files
Ez egy Powershell szkript ami letörli a gépről az ideiglenes fájlokat. Rendszergazdaként futtasd! This is a Powershell script that deletes temporary files. Run this script with Administator priviliges!

Hogyan használd? Felmész a https://github.com/slevente713/RegTweaks/blob/main/AddRunOrModify.reg linkre, és letöltöd ezt a reg fájlt, majd lefuttatod. Ha ez meg van, akkor a letöltött *Debloater Lite.ps1* fájlra nyomsz egy jobbklikket, rámész a Run Or Modify -ra, és kiválasztod a Run As Administrator (Powershell) opciót, és elfogadod a UAC promtot.
How to use it? Go to https://github.com/slevente713/RegTweaks/blob/main/AddRunOrModify.reg and download this file, and run it. Then right-click to the downloaded *Debloater Lite.ps1* then click: "Run Or Modify", and then click: "Rn As Administrator (Powershell)". Then accept the UAC promt, and now, you can run the script.
